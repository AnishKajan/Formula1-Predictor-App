# Formula1-Predictor-App
F1 Predictor App that utilizes Python, Panda, NumPy, Matplotlib, scikit-learn, Joblib, Streamlit

**#Project Structure**
F1-RACE-PREDICTOR/
├── data/ # CSV data generated by fetch_data.py
├── logs/ # Prediction logs from predict.py
├── models/ # Trained ML models and encoders
├── venv/ # Virtual environment (optional)
├── app.py # Streamlit UI app
├── fetch_data.py # Fetches and processes historical race data
├── predict.py # CLI-based predictor and logger
├── train_model.py # Trains models for race predictions
└── README.md # Project documentation
